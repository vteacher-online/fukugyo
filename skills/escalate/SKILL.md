# escalateスキル

督促メールを3回送っても入金がない場合の法的対応をサポートする。

**免責事項: このスキルが生成する文書は参考用のドラフトです。**

📞 無料相談:
- **フリーランス・トラブル110番**（厚労省委託・弁護士対応・匿名OK）
  https://freelance110.mhlw.go.jp/ / 0120-532-110（平日9:30〜16:30）
- 法テラス: 0570-078374

---

## エスカレーションの段階

```
Step 1: 内容証明郵便の送付（e内容証明でブラウザから送れる）
  ↓（2週間後も無応答なら）
Step 2: 法的手続きへ（下記から選択）
  ├─ 支払督促  → 金額上限なし・裁判所不要・オンライン申立可
  └─ 少額訴訟  → 60万円以下のみ・1回の審理で判決・出頭必要
```

---

## コマンド

```
/escalate start [請求書ID]     診断・費用比較・手続き選択
/escalate letter [ID]          内容証明を生成してe内容証明をブラウザで開く
/escalate tokusoku [ID]        支払督促の申立書を生成（金額上限なし）
/escalate shogaku [ID]         少額訴訟の訴状を生成（60万円以下のみ）
/escalate touki                登記簿謄本取得サイトをブラウザで開く
```

---

## `/escalate start` — 診断・費用比較

`python3 scripts/escalate.py start [ID]` を実行する。

支払督促と少額訴訟の費用を並べて比較表示し、どちらを選ぶかユーザーが判断できるようにする：

```
📋 法的手続きの比較
  項目               支払督促    少額訴訟
  請求金額の上限      上限なし    60万円以下
  収入印紙            2,000円     4,000円
  郵便費用等          1,975円     6,500円
  合計概算            4,775円    10,500円
  裁判所への出頭      不要        必要（1回）
  審理期間            書面のみ    1日で判決
  相手の異議        →通常訴訟   なし

💡 選び方の目安:
  支払督促 → 裁判所に行きたくない・60万円超・相手が払う可能性が高い
  少額訴訟 → 60万円以下・1日で決着・証拠が揃っている
```

---

## `/escalate tokusoku` — 支払督促

`python3 scripts/escalate.py tokusoku [ID]` を実行する。

支払督促は**金額上限なし**・**裁判所不要**・**費用は通常訴訟の半額**。

生成する書類（`.fukugyo/escalate/{ID}_支払督促/`）:
- `申立書.txt` — 請求の趣旨・原因・費用・当事者目録
- `準備チェックリスト.md` — 費用・書類・申立方法・手続きの流れ

生成後、督促手続オンラインシステムをブラウザで開くか確認する。

---

## `/escalate shogaku` — 少額訴訟

`python3 scripts/escalate.py shogaku [ID]` を実行する。

60万円超の場合はエラーで利用不可を通知する。

生成する書類（`.fukugyo/escalate/{ID}_少額訴訟/`）:
- `訴状.txt` — 請求の趣旨・原因・証拠方法・附属書類
- `準備チェックリスト.md` — 費用・書類・当日の持ち物・手続きの流れ

---

## 費用計算ロジック

`scripts/escalate.py` の関数:
- `calc_stamp_suit(amount)` — 通常訴訟の収入印紙額（基準値）
- `calc_stamp_tokusoku(amount)` — 支払督促 = 通常訴訟の半額（100円単位切捨て）
- `calc_stamp_shogaku(amount)` — 少額訴訟 = 通常訴訟と同額
